# Music Store SQL Data Analysis

This project uses PostgreSQL to analyze a music store database and extract valuable business insights using SQL queries.

## Tools Used
- PostgreSQL
- SQL
- CSV Dataset

## Files Included
- music_store_queries.sql → Contains all SQL queries
- music_store_dataset.zip → Contains all CSV dataset files

## Analysis Performed
- Sales analysis
- Top customers
- Popular genres
- Best-selling artists
- Revenue by country
